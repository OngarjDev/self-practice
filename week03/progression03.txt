---test_object_parameter_passing.js
ในห้องเรียนผมได้เรียนรู้ว่า 
function myapp() {....} สามารถเขียนย่อเป็น
myapp() {} โดยที่ชื่อมันจะนับเป็น 1 keyด้วย เมื่อเราใส่ func ใน value obj
ผมก็ได้เรียนรู้อีกว่าเราสามารถเรียก myapp()() คือสามารถเรียก func ที่อยู่ใน func ได้ 
ถ้าเราเขียนแค่ชื่อ function เฉยๆ มันจะ console.log ได้ function แต่ถ้าเราใส่ () เพิ่มอีกมันจะreturn เป็น value ของ function
สิ่งที่ผมได้รู้เพิ่มเติมอีกคือ ถ้าเราส่ง object มันจะส่ง ref แทน value แต่ถ้าเราส่ง int ไปให้ function มันจะตีความว่าคือ premative datatype มันจะส่งค่า valueเลย
